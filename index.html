SuiteQL Query Tool

A simple, modern Suitelet to test SQL in NetSuite.

This is a lightweight, easy-to-deploy Suitelet that provides a UI within NetSuite to run SuiteQL queries. It's designed for developers who need a quick way to explore data without the overhead of saved searches or reports.

How to Deploy

  1. Download the suiteql-querytool.js file.
  2. Upload the script to the File Cabinet (Documents > Files > SuiteScripts).
  3. Create a script record (Customization > Scripting > Scripts > New). Deploy the script.
  4. Open the deployment record and access the tool by clicking the link.
